%

% - For students: Education → Skills → Experience → Projects → Leadership
% - For professionals: Experience → Skills → Education → Projects
% - For researchers/academics (CV): Education → Research → Publications → Teaching → Service
%

% 1. Uncomment the additional sections above (Publications, Research, Honors)
% 2. Change "Experience" to "Professional Experience" and "Research Experience"
% 3. Add more detail to each entry (CVs can be 2-3+ pages)
% 4. Include ALL publications, presentations, and grants
% 5. Add teaching experience and course lists if applicable
%
